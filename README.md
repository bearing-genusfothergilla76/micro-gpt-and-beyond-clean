## 🧭 What this is

micro-gpt-and-beyond-clean is a hands-on project that shows how a GPT-style language model works from the ground up.

It starts with plain Python and moves through PyTorch, JAX, and MLX. It also covers basic model serving, so you can see how a model can be used in a real app.

This project is built for learning. It breaks the process into small parts so you can follow along one step at a time.

## 🧠 Main ideas inside the project

The project is focused on a few core ideas:

- A model reads text as tokens
- Tokens turn into vectors
- Attention helps the model focus on related parts
- A transformer stacks these parts together
- Training adjusts weights based on error
- Serving lets the model answer after training

These ideas form the base of many language models.
